# Content Opportunity Scorer

Content Opportunity Scorer is implemented in Phase 8I.

中文描述：
Content Opportunity Scorer 在 Phase 8I 实现。

It scores recommended asset candidates from Phase 8H, prioritises future content assets, provides future adapter routing hints, and prepares for a future Action Pack Router.

中文描述：
它会为 Phase 8H 推荐的候选内容资产评分，排序未来内容资产，提供未来 adapter routing hints，并为未来 Action Pack Router 做准备。

## Boundaries

It does not generate SEO content.

中文描述：
它不生成 SEO 内容。

It does not generate customer-facing copy.

中文描述：
它不生成用户可见文案。

It does not generate Shopify copy.

中文描述：
它不生成 Shopify 文案。

It does not generate email copy.

中文描述：
它不生成 Email 文案。

It does not generate social video scripts or video storyboards.

中文描述：
它不生成社媒视频脚本或视频分镜。

It does not call video factory.

中文描述：
它不调用视频工厂。

It does not generate final visibility scores.

中文描述：
它不生成最终 visibility score。

It does not publish or connect Shopify API.

中文描述：
它不发布，也不连接 Shopify API。

node engines/content_opportunity/content_opportunity_scorer.js friendredlight
