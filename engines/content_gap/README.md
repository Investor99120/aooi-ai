# Content Gap Diagnosis

Content Gap Diagnosis is implemented in Phase 8H.

中文描述：
Content Gap Diagnosis 在 Phase 8H 实现。

It diagnoses content gaps from manual AI visibility observations, identifies missing brand-owned educational assets, identifies claim-safe content needs, identifies future asset candidates, and prepares for a future Content Opportunity Scorer.

中文描述：
它基于人工 AI visibility observation 诊断内容缺口，识别缺失的品牌自有教育内容资产、claim-safe 内容需求和未来资产候选，并为未来 Content Opportunity Scorer 做准备。

## Boundaries

It does not generate SEO content.

中文描述：
它不生成 SEO 内容。

It does not generate customer-facing copy.

中文描述：
它不生成用户可见文案。

It does not generate content opportunity scores.

中文描述：
它不生成内容机会评分。

It does not generate final visibility scores.

中文描述：
它不生成最终 visibility score。

It does not publish or connect Shopify API.

中文描述：
它不发布，也不连接 Shopify API。
